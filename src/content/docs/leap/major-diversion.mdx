## 信息技术学域（二）

不出意外的话会在大一结束后分流，学域内软件工程、网络工程、集成电路设计与集成系统、电子科学与技术四个专业任选。

软件工程和网络工程属于计算机类专业，软件学院开办；集成电路设计与集成系统和电子科学与技术属于电子信息类专业，集成电路学院开办。

软件工程和网络工程同属计算机类，均为工学专业。根据往年培养方案，本科学习内容差别不大，考研的话也没啥区别，大类通识课都一样，大二下以及大三的专业选修课会有区别。在专业课上，软件工程会开设移动应用开发、嵌入式软件设计、机器人视觉识别技术等课程，网络工程会开设网络信息安全、密码学、网络协议栈分析与设计等课程。两专业学费相同，毕业证上专业有区别。

集成电路学院的两个分流方向通常简称为设计与器件。集成电路设计与集成系统便是人们所熟知的 IC 设计方向，电子科学与技术则偏向于对器件材料的研究包括工艺流程等等，也有非常好的发展前景。用最便于理解的话（当然并不全面准确）来说，集成电路设计与集成系统就是设计芯片，电子科学与技术就是制造芯片。根据集成电路学院往年的培养方案，这两个专业分流后其实区别也不会很大，课程相似度也很高，往往只是必修与选修的区别，因此分流时建议主要遵循个人意愿，不存在“谁比谁好”这样的说法。顺带一提的是，23 级及以前，集成电路学院是在大二结束后才进行分流，分流后的课程基本集中在了大三上学期这一个学期，因此在外校学生与老师眼里，这两个分流方向并没有太大的区别，分流到设计未来也可以做器件，分流到器件未来也可以做设计，两者是互通的。

由于学域书院制培养方案尚未公布，实际与往年略有出入属于正常情况，以上内容仅供参考。

<Aside>
  专业确定之后是按照专业方案进行课程的修读，软件专业同学选软件的模块选修，网络专业同学选网络的模块选修，因培养方案是相互独立的，修读非自己专业的模块选修课程不认定为自己的模块学分且不能进行置换。如因感兴趣不计较是否计入毕业总学分，在课容有余量的情况下可以选非本专业的选修课学习。
</Aside>
### 软件工程专业介绍

软件工程专业代码：080902，属于 **0809 计算机类**。授予**工学学士学位**，一级学科为**软件工程**。

简单理解软件工程：软件工程就是研究构建和维护有效的、实用的、和高质量的软件的理论、方法和工具。

软件工程定义：用计算机科学、数学、逻辑学及管理科学等原理，开发软件的工程。软件工程借鉴传统工程的原则、方法，以提高质量、降低成本和改进算法。

- 计算机科学、数学用于构建模型与算法
- 工程科学用于制定规范、设计范型(paradigm)、评估成本及确定权衡
- 管理科学用于计划、资源、质量、成本等管理。

专业成立于 2003 年，依托大连理工大学软件学院（教育部和国家计委联合批准成立的首批 35 所国家级示范性软件学院）建设。专业所依托的软件工程学科入选省双一流重点建设学科，第五次全国高校学科评估为 A，软件工程学科入选中国一流学科排名软件工程全国 A+。

必修课：计算机网络、操作系统、数据库系统、编译技术、Java 高级编程与应用 / Python 与数据分析（二选一）、网络综合实验、软件工程、实习实训、人工智能基础、系统分析与设计、IT 从业素养、专业方向课程设计

各个专业方向：

- 软件开发与测试（方向/模块）
  - 培养目标：全面掌握软件工程的基本理论；熟练应用主流软件平台及管理工具；注重工程实践能力和外语应用能力
  - 开设课程
    - 技术类：可视化原理与实现技术，J2EE 高级编程，移动应用开发，开源软件基础，设计模式，软件安全基础
    - 工程类：工程经济学，软件项目管理，软件测试与质量保证，移动应用测试
  - 就业方向：适应软件结合其他专业领域的复合人才需求，各种 IT 技术企业，具有持久的人才发展潜力，最大限度地成为企业的中高端技术和管理人才。就业领域遍布于软件系统层、应用层的研发。
- 智能系统技术（方向/模块）：
  - 培养目标：以就业为导向，能够解决复杂工程问题的复合型、应用型软件工程技术人才。
  - 开设课程：智能嵌入式系统原理与设计，智能控制系统，智能嵌入式操作系统，嵌入式软件形式化设计与验证，智能嵌入式软件设计方法，FPGA 设计与应用，Python 人工智能程序设计，智能网络管理系统
- 金融科技（方向/模块）（原金融信息化）：
  - 培养目标：培养具有金融领域前沿知识，掌握软件行业前沿技术，具备国际化视野的复合型卓越人才。
  - 开设课程：金融学基础，数据安全管理，证券交易实务，互联网金融技术，金融信息系统与智能金融，区块链与数字货币，金融科技设计实践
- 大数据技术与应用（方向/模块）：
  - 培养目标：掌握数据科学扎实的基础理论和专业知识；掌握大数据采集、存储、处理、分析等技术，具备大数据应用系统的设计和开发能力；胜任大数据分析挖掘、大数据系统开发等技术领域的工作
  - 开设课程：云计算与大数据基础，机器学习，大数据计算技术，数据挖掘技术，数据可视化与案例分析
  - 方向特色：校企合作、案例驱动、实践创新
- 水下智能机器人（方向/模块）：
  - 培养目标：具备机器人常识、定位于人工智能专业人才的培养
  - 开设课程：机器人操作系统，水下机器人结构与控制，机器人视觉识别技术，机器学习与深度学习，机器人可视化技术
  - 培养模式：“产+学+研+赛”四位一体的激励式人才培养模式

### 网络工程专业介绍

网络工程专业代码：080903，属于 **0809 计算机类**。授予**工学学士学位**，一级学科为**软件工程**。

培养目标：聚焦国家网络重大需求，培养创新、复合、 厚植爱国情怀的网络工程技术和管理人才。具备网络与通信宽厚理论知识基础，掌握现 代网络工程前沿技术，具备在网络安全、物联网、工业互联网等行业和领域从事工程科学研究、新技术研发、工程创新设计等方面 能力的研究型创新人才。。以产教融合，科教，将网络安全、大数据、 云计算、物联网相融合；同 IBM、微软、腾 讯、华为等行业领军企业联合，强化实训， 培养卓越人才，建一流课程培养一流学生。

必修课：计算机网络、操作系统、数据库系统、高级 C 语言及网络编程技术、Java 高级编程与应用 / Python 与数据分析（二选一）、网络综合实验、软件工程、网络信息安全、人工智能基础、网络协议栈分析与设计、IT 从业素养、专业方向课程设计

两个专业方向：网络空间安全、物联网工程（专业总人数 120 人左右）。

#### 1. 网络空间安全

- **培养目标**：培养具备扎实网络安全理论基础，能够应对复杂网络攻击与防御问题的高素质应用型、复合型网络安全人才。学生将掌握网络安全的基本技术，熟悉网络攻防对抗，具备独立设计和实现网络安全策略、解决实际网络安全问题的能力。
- 开设课程：
  - 互联网+、系统安全理论及应用、信号传输与处理、Web 技术与应用、无线网络安全技术、网络攻防与实践

#### 2. 物联网工程

- **培养目标**：培养能够设计和实现物联网系统的应用型技术人才，具备嵌入式系统开发、无线通信技术、物联网协议、传感器网络、智能硬件设计等领域的专业能力，能在物联网应用开发、系统集成及管理等工作中胜任重要工作。
- 开设课程：
  - 物联网感知关键技术、移动计算与应用、物联网编程技术、物联网大数据智能处理、云计算、工业物联网技术

升学情况：2019 级学生 119 人，保研 20 人。最近四年升学人数超 1/3，其中境内升学 100 人，境外 43 人。

## 国际信息与软件学院

<Aside>
  由于24级新增专业物联网工程，且全校推广学域制办学，因此分流比例和政策都可能有变化，以下内容仅供参考，请知悉。
</Aside>

软国往年在大二下学期结束后分成软件工程和数字媒体**技术**两个专业，保研名额按比例均匀分配。分流名额由于近几届招生人数均不同则难以下定论。可以作为参考的是：22 级招生人数 210 人，其中软工名额 90 人，数媒 120 人，最后一个分流进入软件工程专业的排名为 135/207。

专业分流的参评课程是保研参评课程的大一大二部分。分流时先选择志愿，之后按绩点排名分流，名额满则超出名额的分至另一专业。

软件工程偏软件开发一点，数媒会多点游戏制作和视觉方面的东西，实际学的差距不大，但是软件工程排名靠前的人会更多一点，保研绩点线会稍高一点。

2+2 也需要参与专业分流，因为会影响到在立命馆毕业后拿到的大工毕业证上面的就读专业名称。
