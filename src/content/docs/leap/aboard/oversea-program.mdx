## 海外 IT 实训项目

国际信息与软件学院（软国）往年设有海外 IT 实训/实习项目，往年该课程对软国学生为**必修**。2026 年起该项目改为**非必修**，且日本（立命馆大学）方向的选课已取消，目前仅保留英国布里斯托大学（Bristol）方向，需要在大二上学期通过选课系统选择。

> **提示**：往年参加该项目的同学较多，今年政策有变动，选课前请留意学院最新通知，确认是否仍有选课需求。

### 往年项目参考（已停办/调整方向）

- 日本立命馆大学（2+2 项目）：前两年在大工就读，后两年在立命馆大学情报理工学部就读，可获得两校学位。2026 年起日本方向不再通过该实训项目选课。
- 英国布里斯托大学 3+1 联合培养：大工读 3 年 + 布里斯托读 1 年，可获得大工本科 + 布里斯托授课型硕士双学位。

详见[求实书院](/leap/major/qiushi-college)页面中的 2+2 与 4+0 学制说明。
