集成电路学院的两个分流方向通常简称为设计与器件。集成电路设计与集成系统便是人们所熟知的 IC 设计方向，电子科学与技术则偏向于对器件材料的研究包括工艺流程等等，也有非常好的发展前景。用最便于理解的话（当然并不全面准确）来说，集成电路设计与集成系统就是设计芯片，电子科学与技术就是制造芯片。根据集成电路学院往年的培养方案，这两个专业分流后其实区别也不会很大，课程相似度也很高，往往只是必修与选修的区别，因此分流时建议主要遵循个人意愿，不存在“谁比谁好”这样的说法。顺带一提的是，23 级及以前，集成电路学院是在大二结束后才进行分流，分流后的课程基本集中在了大三上学期这一个学期，因此在外校学生与老师眼里，这两个分流方向并没有太大的区别，分流到设计未来也可以做器件，分流到器件未来也可以做设计，两者是互通的。

## 软件工程

软件工程专业代码：080902，属于 **0809 计算机类**。授予**工学学士学位**，一级学科为**软件工程**。

简单理解软件工程：软件工程就是研究构建和维护有效的、实用的、和高质量的软件的理论、方法和工具。

软件工程定义：用计算机科学、数学、逻辑学及管理科学等原理，开发软件的工程。软件工程借鉴传统工程的原则、方法，以提高质量、降低成本和改进算法。

- 计算机科学、数学用于构建模型与算法
- 工程科学用于制定规范、设计范型(paradigm)、评估成本及确定权衡
- 管理科学用于计划、资源、质量、成本等管理。

专业成立于 2003 年，依托大连理工大学软件学院（教育部和国家计委联合批准成立的首批 35 所国家级示范性软件学院）建设。专业所依托的软件工程学科入选省双一流重点建设学科，第五次全国高校学科评估为 A，软件工程学科入选中国一流学科排名软件工程全国 A+。

必修课：计算机网络、操作系统、数据库系统、编译技术、Java 高级编程与应用 / Python 与数据分析（二选一）、网络综合实验、软件工程、实习实训、人工智能基础、系统分析与设计、IT 从业素养、专业方向课程设计

各个专业方向：

- 软件开发与测试（方向/模块）
  - 培养目标：全面掌握软件工程的基本理论；熟练应用主流软件平台及管理工具；注重工程实践能力和外语应用能力
  - 开设课程
    - 技术类：可视化原理与实现技术，J2EE 高级编程，移动应用开发，开源软件基础，设计模式，软件安全基础
    - 工程类：工程经济学，软件项目管理，软件测试与质量保证，移动应用测试
  - 就业方向：适应软件结合其他专业领域的复合人才需求，各种 IT 技术企业，具有持久的人才发展潜力，最大限度地成为企业的中高端技术和管理人才。就业领域遍布于软件系统层、应用层的研发。
- 智能系统技术（方向/模块）：
  - 培养目标：以就业为导向，能够解决复杂工程问题的复合型、应用型软件工程技术人才。
  - 开设课程：智能嵌入式系统原理与设计，智能控制系统，智能嵌入式操作系统，嵌入式软件形式化设计与验证，智能嵌入式软件设计方法，FPGA 设计与应用，Python 人工智能程序设计，智能网络管理系统
- 金融科技（方向/模块）（原金融信息化）：
  - 培养目标：培养具有金融领域前沿知识，掌握软件行业前沿技术，具备国际化视野的复合型卓越人才。
  - 开设课程：金融学基础，数据安全管理，证券交易实务，互联网金融技术，金融信息系统与智能金融，区块链与数字货币，金融科技设计实践
- 大数据技术与应用（方向/模块）：
  - 培养目标：掌握数据科学扎实的基础理论和专业知识；掌握大数据采集、存储、处理、分析等技术，具备大数据应用系统的设计和开发能力；胜任大数据分析挖掘、大数据系统开发等技术领域的工作
  - 开设课程：云计算与大数据基础，机器学习，大数据计算技术，数据挖掘技术，数据可视化与案例分析
  - 方向特色：校企合作、案例驱动、实践创新
- 水下智能机器人（方向/模块）：
  - 培养目标：具备机器人常识、定位于人工智能专业人才的培养
  - 开设课程：机器人操作系统，水下机器人结构与控制，机器人视觉识别技术，机器学习与深度学习，机器人可视化技术
  - 培养模式：“产+学+研+赛”四位一体的激励式人才培养模式
