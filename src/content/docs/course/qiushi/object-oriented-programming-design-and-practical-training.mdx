## 课程简介

软件学院在大一小学期开设 C++ 程序设计课程，通常为期约两周。课程主要内容包括面向对象编程（OOP）和 EasyX 图形库，最终大作业要求用 EasyX 实现一个可运行的小游戏，将 OOP 知识与图形库结合。

## 课程要求

- 课程内容：面向对象编程（OOP）+ EasyX 图形库
- 大作业：使用 EasyX 实现一个可运行的小游戏

## 注意事项

- 大作业提交形式往年通常为**录制视频**即可；部分老师（如 ck）可能要求当堂展示，如有疑问建议提前向授课老师确认。
- 课程结束后一般即可离校，后续无强制在校安排。
- 有同学反映大二学年专业课与代码实践关联度较低，建议珍惜小学期 C++ 课程的编程机会。
