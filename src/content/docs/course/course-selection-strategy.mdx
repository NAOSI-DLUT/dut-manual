## 预选阶段攻略

在预选阶段所有课程不限人数，选课不分时间先后，因此不需要着急，在预选的几天内合理安排时间选课即可。

预选阶段进行的是必修和公共选修课的选择。开发区校区的体育课在正选阶段进行抢课，预选阶段的体育课只是随机分配的占坑课程，在预选阶段结束后自动退掉，无需在意。

## 意愿值

在预选阶段，每个人会被分配到 **100 意愿值**。预选结束后会启动抽签机制：你选这门课用 n 个意愿值，相当于在这门课的桶里投了 n+1 个签，系统会对所有签进行随机遍历。如果某个签对应的同学没选上这门课，则将其加入成功选课列表；如果该同学已选上课，则继续进行下一轮遍历，直到成功选课列表的 size 与总人数相等。

也就是说，用的意愿值越多，选上课的概率越大，但即使给 100 个意愿值，概率也不会等于 100%；用的意愿值越少，选上课的概率越小，但即使给 0 个意愿值，概率也不会等于 0%。另外，如果课程没被选满，那么给 0 个意愿值也能选上课。

对于班级课表里面的必修课来说，意愿值分配 **0** 即可，如果实在担心掉课，可以分配 **1-5 意愿值**。意愿值主要用在选修课和劳动课等的抽选，由于开发区劳动 2 课程容量极其有限，美育课一般只有音乐鉴赏一个选项，因此抢这门课程很多时候 100 意愿值也难选上。请珍惜你选上的选修课和意愿值。

## 必修课

对于必修课，通常直接按照班级课表选课，并且意愿值给 **0** 即可。如果认为班级课表里分配的老师可能有争议，且所在学院允许跨教学班选课，可以选择同时间的另一个教学班老师的同名课程，并分配一定的意愿值。

如果你的课程不能在预选阶段跨教学班选课，那么预选可以不用选他，等正选的时候直接选其他老师的课。因为要避免时间冲突，如果预选选上了，正选的时候要先退课才能选，会慢人一步。

## 通识/选修课

多数学院本科生培养方案规定需要修读：

- 全校通识课
- 劳动 2 课程
- 《国家安全教育》课程
- 创新创业课程
- “四史类”课程
- “美育类”课程
- 《经济学原理》及《现代工程导论》两门课中选修一门

这些课程在毕业之前需要修读完毕。开发区校区除国家安全教育作为网络通识课之外，其他几类课程的开课容量和时间都比较有限。因此建议从大一开始就选择这些课程，以免后期选不上或者必修课时间覆盖掉这些课程。

关于 **四史类** 课程，据同学反馈该课程**期期撞课**（与必修课冲突概率高），选课前务必留意培养方案中对该课程的学期要求，提前做好规划。

如果因时间冲突实在选不上，可以私聊教务老师帮忙协调。教务老师工作日工作时间内在 QQ 上会回复，通常几小时内可收到回复，也可以直接前往教务办公室当面沟通。

关于 **美育类** 课程，常见选项包括**音乐鉴赏**。《数据时代的艺术表达》属于**综合素质类**课程，不在美育类模块内。该课程无统一期末考试，考勤为老师随机点名（通过发纸写题目的方式进行），且结课有大作业（如手绘邮票设计，用纸笔手绘完成，非电脑作图）。

24 级通识课要求为：经济学/现代工程导论 + 美育课 + 任意一门其他通识课（即三门即可满足要求），并非所有类别都要修读。建议根据本人培养方案确认具体要求后再选课。

《经济学原理》在开发区校区部分学期可能不开放选课，主校区和盘锦校区通常可选。若培养方案要求在《经济学原理》和《现代工程导论》中二选一，选课前请先确认本校区是否有开课。

## 必修课时间冲突

如果你在选课时发现两门课程都像是“必修”，但系统提示存在时间冲突，不要急着盲选，先做这三步：

1. 回到培养计划里确认两门课是否都属于当前学期必须完成的课程。
2. 在选课系统里查看课程名称后的开课时间，判断这门课是不是后续学期还会继续开设。
3. 再结合班级课表确认冲突是不是教务排课导致的同一时段冲突。

一般来说，如果其中一门课在后续多个学期还会继续开，而另一门课只在某个固定学期开放，那么优先保留开课窗口更窄的那门课。以软件学院（软国）为例，1-5 班小学期课程与“改开史”（改革史）冲突，6-10 班与“创新”冲突。两种情况下判断依据一致：程序课只在大一小学期开设，改开史/创新课程后续学期还会继续开，因此优先保留程序课，另一门放到后续学期补修更稳妥。

如果两门课都不确定是否能后续补选，或者都显示为当前培养计划中的关键课程，建议尽快咨询辅导员、教学秘书或对应课程老师，不要仅凭意愿值和抢课结果硬赌。

## 注意事项

1. **《国家安全教育》课程**建议直接在课比较少、时间充裕的大一下学期选课。它作为网络通识课容量无限且不占据正课时间，下个学期考试前将网络平台上的视频看完并按时在电脑上参加考试即可。
2. **非必修课**选完后如果忘了去上，需要白交学费，成绩单上会留下 0 分记录，但据同学反馈一般不会影响保研等其他事项。不过仍建议尽量避免这种情况。
3. 软国有意向参加“2+2”赴日学习的同学，在赴日前必须完成“国家安全教育”课程的修读。
4. 网络通识课均为通识类扩展课程，有“尔雅通识教育网络课”（课程代码以“y”结尾）和“东西部联盟通识教育网络课”（课程代码以“z”结尾）。该类课程均为网络授课，不安排具体时间地点。请选择此类课程的同学认真检查选课结果，并在规定的学习平台进行注册、学习及考核。
5. **小学期课程冲突**：部分同学反映创业、四史与小学期必修实践课存在冲突。可参考往届经验，在同一时间段的实践课中选择其他教学班来避开冲突。
6. **小学期课程实际情况**：据同学反馈，大二小学期多数课程出勤要求较严，基本不存在能提前离校的课，大部分老师每节课都会点名。选课前不宜抱侥幸心理。此外，选课系统中填写的老师不一定是实际上课的老师，以实际情况为准。
7. **创新创业课程**：两门创新创业课程均需修读。签到通常以问卷二维码形式进行。部分班级最终以小组大作业形式评定成绩。
8. **四史课程**：中共党史（2026 年数据）也为机考开卷形式，占比 50%，可带教材和打印材料，题型较固定，建议找老师要复习 PPT 直接背诵。
9. **跨学院选课学分认定**：选课时如果选择了其他学院开设的课程（教学班不属于自己所在学院），学分认定存在不确定性。如需跨学院选课，建议提前咨询本学院教务老师确认学分是否可认定。
10. **0 学分课程**（如部分“专创融合”课程）：此类课程不纳入毕业学分统计，不影响最终成绩，也不需要额外缴纳学分制学费。成绩单上可能会显示 0 分，但据同学反馈不会拉低平均分，无需补考或重修。
