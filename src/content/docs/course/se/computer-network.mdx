## 学习建议

据多位同学反馈，计网是不少同学**挂的第一门课**，难度较高，建议认真对待。

### 复习资料

部分教师（如李晓晨老师）的录播可在**学习通**相应章节中查看。大工云盘中也有同学上传的计网录播资源，可以留意群内共享链接。

有同学分享了各章知识要点的复习 PPT，期末前建议结合老师的复习范围进行针对性复习。

### 注意事项

软件学院（软国）和软件学院的计网课程为**中文授课**，有同学反映其难度比软国课程更高，备考时建议以中文教材和课件为主。
